# Чат-бот для абитуриентов УрФУ (программа «Правовое обеспечение национальной безопасности»)

Этот Telegram-бот создан для помощи абитуриентам, интересующимся поступлением на программу «Правовое обеспечение национальной безопасности» (ПОНБ) Уральского федерального университета. Бот предоставляет информацию о юридических направлениях, процессе поступления, профессиональной ориентации и студенческой жизни.

## Основные возможности

### 📘 Блок 1: Поступление и программы
- **Юридические направления в УрФУ**: список всех юридических специальностей (бакалавриат и специалитет) с количеством мест и стоимостью обучения.
- **Чем отличаются направления**: сравнение программ бакалавриата и специалитета, различия по специализации.
- **Кем работать после ПОНБ**: перечень возможных мест работы выпускников.
- **Особенности ПОНБ в УрФУ**: ссылка на интервью с руководителем программы.
- **Как поступить на ПОНБ**: экзамены (ЕГЭ), индивидуальные достижения, сроки подачи документов и способы подачи.

### ⚖️ Блок 2: Выбор профессии
- **Видео от руководителя программы**: прямая ссылка на YouTube.
- **Отзывы выпускников об обучении**: ссылка на видео (Rutube).
- **Знакомство с представителями профессий**: подробное описание практико-ориентированного обучения и три кнопки-ссылки на интервью с представителями ФСБ, прокуратуры и адвокатуры.
- **Тест «Тебе подходит профессия юриста?»**: интерактивный тест из 13 вопросов с выводом результата (4 типа: прирождённый юрист, гибкий гуманитарий, свободный художник, идейный госслужащий).
- **Другие тесты для самоопределения**: ссылки на внешние профориентационные тесты (Ikigai, Testometrika).

### 🎓 Блок 3: Жизнь студента
- **Корпуса и учебные здания**: адреса основных корпусов и ссылка на видеообзор.
- **Физкультура и спорт**: описание выбора спортивных дисциплин и ссылка на видео о спортивных объектах.
- **Общежития**: ссылка на сайт студенческого городка.
- **Студенческая жизнь**: четыре кнопки-ссылки на мероприятия (встреча в МВД, конкурс судебных процессов, интервью с преподавателем, день юриста).
- **Полезные интернет-порталы**: каталог социальных сетей программы ПОНБ, Института экономики и управления (ИнЭУ) и групп для абитуриентов.

### Общая навигация
- Кнопка «🏠 Главное меню» возвращает в начальное меню.
- Кнопка «◀️ Назад» ведёт на предыдущий уровень вложенности.
- Все видео реализованы как кнопки-ссылки (без встроенного воспроизведения внутри бота), что упрощает хостинг и не требует загрузки видеофайлов.

## Технологии

- **Python 3.11+**
- **aiogram 3.x** – асинхронный фреймворк для Telegram Bot API
- **python-dotenv** (опционально, для локальной разработки)
- Хостинг: рекомендуется **bothost.ru**, **Render** или любой другой, поддерживающий long polling
